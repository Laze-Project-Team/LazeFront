ロゴガイドライン

LazeのロゴはLaze Project Teamが所有し、以下の規定のもとに再利用及び改変を許可します。

使用条件

ロゴの帰属がLaze Project Teamである旨を適切な方法で記載してください。

ロゴの改変は以下の場合に限って許可されます。

  • アスペクト比を変えないリサイズ（ただし最小サイズは16x16）
  • デザインの都合でのLazeのイメージを損なわない範囲での色の変更（例えば白黒にするなど）

改変が許可されるかどうか疑問に思った場合には discordコミュニティ またはメール（lazelang.official@gmail.com） を通してLaze Project Teamにお知らせください。

ロゴの種類

Lazeのロゴは以下に示すようにLazeの文字が入っているものと入っていないものの2種類があります。状況やデザインに合わせて適宜使い分けてください。

ブランドカラー

Lazeのブランドカラーはロゴを構成する二つの色です。以下に具体的な色を示します。ブランドカラーはLazeのイメージを形作る上で重要な役割を果たします。

Laze Main Color

HEX ― #EBA059

RGB ― R235 G160 B89

CMYK ― C0% M32% Y62% K8%

Laze Spare Color

HEX ― #F2BD8B

RGB ― R242 G189 B139

CMYK ― C0% M22% Y43% K5%

禁止事項

以下の事項は禁止されています。

  • 使用条件を守らないでロゴを使用すること
  • Lazeのブランドカラーとかけ離れた色のロゴを使用すること
  • Lazeと全く関連性のないコンテンツにロゴを使用すること
  • その他Lazeのイメージを低下させるようなロゴの使用をすること
